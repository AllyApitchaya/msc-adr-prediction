# msc-adr-prediction

Machine learning for predicting adverse drug reactions (ADRs) from drug molecular features and public pharmacovigilance data.

This repository contains the code for an MSc dissertation submitted to the University of Edinburgh (MSc Data Science, Technology and Innovation).

## Overview

Adverse drug reactions are a major cause of patient harm. This project develops a machine learning approach to predict drug-ADR associations by combining:

- Molecular structure features of drugs, retrieved from PubChem
- Pharmacovigilance signals from the FDA Adverse Event Reporting System (FAERS)

As a proof of concept, the study focuses on antidiabetic drugs (ATC class A10).

## Data sources

| Source | Description | Access |
|--------|-------------|--------|
| PubChem | Drug molecular structures (SMILES) | Public, CC0 |
| FAERS | Adverse event reports (2020-2022) | Public, FDA |

Raw and processed data files are not included in this repository, due to file size and data redistribution considerations. All data can be regenerated by running the notebooks in order.

## Repository structure

- notebooks/01_data_collection.ipynb - Data retrieval (PubChem + FAERS)
- notebooks/02_preprocessing.ipynb - Cleaning, splitting, negative sampling
- .gitignore
- LICENSE
- README.md

## Pipeline

1. Data collection (01_data_collection.ipynb): retrieves molecular structures from PubChem and adverse event reports from FAERS, then combines them into a single drug-ADR dataset.

2. Preprocessing (02_preprocessing.ipynb): removes non-ADR noise terms, splits the data temporally into train/validation/test sets to prevent leakage, and generates balanced negative samples.

## How to reproduce

The notebooks are designed to run on Google Colab.

1. Run 01_data_collection.ipynb to download and assemble the dataset.
2. Run 02_preprocessing.ipynb to produce the model-ready train, validation, and test splits.
